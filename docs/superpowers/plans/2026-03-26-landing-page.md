# ANERTiC Landing Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a public marketing landing page at `web/anertic.com/` using React Router 7 SSR with Tailwind CSS 4, featuring an animated AI orb hero, feature cards, pricing, and contact form.

**Architecture:** Single-page SSR app. One route (`home.tsx`) composes all section components. Each section is its own component file. A `ScrollReveal` wrapper handles scroll-triggered animations via IntersectionObserver. No backend integration — static marketing content only.

**Tech Stack:** React Router 7 (SSR), Vite 7, Tailwind CSS 4, Plus Jakarta Sans, pnpm

**Spec:** `docs/superpowers/specs/2026-03-26-landing-page-design.md`

**Visual reference:** `.superpowers/brainstorm/52347-1774493253/content/page-layout-v6.html`

### Task 3: ScrollReveal Component

**Files:**
- Create: `web/anertic.com/app/components/scroll-reveal.tsx`

- [ ] **Step 1: Create scroll-reveal.tsx**

A reusable wrapper component that uses IntersectionObserver to add a `.show` class when the element enters the viewport. Supports staggered `delay` prop for grid children.

```tsx
import { useEffect, useRef, type ReactNode } from 'react'
import { cn } from '~/lib/utils'

export function ScrollReveal({
  children,
  className,
  delay = 0,
}: {
  children: ReactNode
  className?: string
  delay?: number
}) {
  const ref = useRef<HTMLDivElement>(null)

  useEffect(() => {
    const el = ref.current
    if (!el) return

    const observer = new IntersectionObserver(
      ([entry]) => {
        if (entry.isIntersecting) {
          el.classList.add('show')
          observer.unobserve(el)
        }
      },
      { threshold: 0.12, rootMargin: '0px 0px -30px 0px' },
    )

    observer.observe(el)
    return () => observer.disconnect()
  }, [])

  return (
    <div
      ref={ref}
      className={cn(
        'motion-safe:translate-y-7 motion-safe:opacity-0 motion-safe:transition-all motion-safe:duration-700',
        'motion-safe:[transition-timing-function:cubic-bezier(0.16,1,0.3,1)]',
        'motion-safe:[&.show]:translate-y-0 motion-safe:[&.show]:opacity-100',
        className,
      )}
      style={{ transitionDelay: `${delay}ms` }}
    >
      {children}
    </div>
  )
}
```

```dockerfile
FROM node:20-alpine AS base
RUN corepack enable && corepack prepare pnpm@latest --activate

FROM base AS dependencies
COPY web/anertic.com/package.json web/anertic.com/pnpm-lock.yaml /app/
WORKDIR /app
RUN pnpm install --frozen-lockfile

FROM base AS build
COPY web/anertic.com/ /app/
COPY --from=dependencies /app/node_modules /app/node_modules
WORKDIR /app
RUN pnpm build

FROM base AS prod-dependencies
COPY web/anertic.com/package.json web/anertic.com/pnpm-lock.yaml /app/
WORKDIR /app
RUN pnpm install --frozen-lockfile --prod

FROM base AS runtime
COPY --from=build /app/build /app/build
COPY --from=build /app/package.json /app/
COPY --from=prod-dependencies /app/node_modules /app/node_modules
WORKDIR /app
ENV NODE_ENV=production
ENV PORT=8080
EXPOSE 8080
CMD ["pnpm", "start"]
```
